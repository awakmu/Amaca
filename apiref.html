Api Reference¶

Intro¶

The file amaca.h is used to include all the functions and data types declarations. It’s enough to:

#include <amaca.h>

in each source file to start using Tofu. All the functions are prefixed by amaca_.

The templates are just plain text strings or files containing blocks of Lua code. The code blocks have to be enclosed between {{ and }}.

String¶

char *amaca_eval(const char *template, int argc, ...);

Evaluate the given template string, passing to it the following argc key-value pairs.

New in version 0.5.

char *amaca_veval(const char *template, int argc, va_list args);

Equivalent to amaca_eval except that it is called with a va_list instead of a variable number of arguments.

New in version 0.5.

File¶

char *amaca_eval_file(const char *filename, int argc, ...);

New in version 0.5.

char *amaca_veval_file(const char *filename, int argc, va_list args);

Equivalent to amaca_eval_file except that it is called with a va_list instead of a variable number of arguments.

New in version 0.5.

File descriptor¶
